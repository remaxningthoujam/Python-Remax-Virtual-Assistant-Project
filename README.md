# 🤖 Python Chatbot Project
### A simple chatbot project built using Python. This chatbot can chat with users, tell joke, give motivational quotes, perform calculations, show date & time, save chat history, open websites, and support voiec features using os.system().
## 🚀 Features
### 💬 Chat Features
- Greeting responses
- Motivational quotes
- "How are you?" replies
- Chatbot name response
- joke response
- Bye/exit response
- Unknown message handling
### 🧮 Calculator Features
- Addition
- Subtraction
- Multiplication
- Dividion
- Percentage calculation
### 🕛 Utility Features
- Show current DATE
- Show current TIME
- Save chat history in text file
### 🌐 Browser Features
Using Python ```webbrowser``` and ```os``` module:
- Open Youtube
- Open Google
### 🔊 Voice Features
Using Python ```os.system()```
- Chatbot voice output
- Speak output responses through system voice
## 🛠️ Technologies Used
- Python
- random
- datetime
- os
- webbrowser
